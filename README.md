# Icon Finances Analyzer

## Overview
Icon Finances Analyzer is a Java-based application designed to analyze users and the infrastructure of a company. It provides a modular framework for evaluating performance, generating insights, and creating structured reports.

## Features
- User analysis with performance metrics
- Infrastructure monitoring and status reporting
- Modular architecture for easy extension
- CSV report generation
- Random data generator for testing purposes
- Logging and utility modules

## Project Structure
- `src/` — Contains all source code organized by models, services, utils, reports, and database
- `models/` — Data models including User, Department, and Infrastructure
- `services/` — Business logic for user and infrastructure analysis
- `reports/` — Report generation modules
- `utils/` — Helper classes for data generation and logging
- `database/` — Database connection and initialization modules
- `Main.java` — Entry point of the application
